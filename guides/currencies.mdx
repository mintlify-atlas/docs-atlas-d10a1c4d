The Stake Engine API represents all monetary amounts as integers scaled by a fixed multiplier. This page explains the encoding, the helper functions for parsing and displaying amounts, and the full list of supported currencies.

## Amount encoding

All amounts from the API are integers. The client uses a fixed multiplier of **1,000,000** to encode decimal values as integers.

| API value | Decimal value |
|----------:|---------------|
| `1000000` | `1.00` |
| `500000` | `0.50` |
| `10000000` | `10.00` |
| `100` | `0.0001` |

Use `ParseAmount` to convert an API integer to a JavaScript `number`:

```typescript
import { ParseAmount } from 'stake-engine';

// Convert API integer to decimal
ParseAmount(1_000_000); // => 1
ParseAmount(1_500_000); // => 1.5

// Convert a decimal bet to an API-encoded amount
const betAmount = 2.50 * 1_000_000; // => 2_500_000
```

`ParseAmount` simply divides by `1,000,000`:

```typescript
const ParseAmount = (val: number): number => {
  return val / 1_000_000;
};
```

<Note>
  When passing amounts to `Play()`, provide the API-encoded integer directly (e.g. `1_000_000` for $1.00), not the decimal value. Bet levels returned by `Authenticate()` are already API-encoded.
</Note>

## DisplayAmount

`DisplayAmount` formats a `Balance` object into a human-readable string with currency symbol, correct decimal precision, and locale-aware number formatting.

```typescript
import { DisplayAmount } from 'stake-engine';
import type { Balance } from 'stake-engine';

const balance: Balance = { amount: 10_500_000, currency: 'USD' };

DisplayAmount(balance);                                   // => "$10.50"
DisplayAmount(balance, { removeSymbol: true });           // => "10.50"
DisplayAmount(balance, { decimals: 4 });                  // => "$10.5000"
DisplayAmount(balance, { trimDecimalForIntegers: true }); // => "$10.50" (no change — not a whole number)

const wholeBalance: Balance = { amount: 5_000_000, currency: 'USD' };
DisplayAmount(wholeBalance, { trimDecimalForIntegers: true }); // => "$5"
```

### Options

| Option | Type | Description |
|--------|------|-------------|
| `removeSymbol` | `boolean` | Removes the currency symbol from the output. Useful for inputs or numeric-only displays. |
| `decimals` | `number` | Forces a specific number of decimal places, overriding the currency default. |
| `trimDecimalForIntegers` | `boolean` | When the amount is a whole number, removes the decimal portion entirely (e.g. `$2.00` → `$2`). Useful for displaying bet levels. |

All options are optional. When omitted, the currency's default decimal precision is used.

### Symbol placement

Some currencies place the symbol after the number. `DisplayAmount` handles this automatically:

```typescript
const dkk: Balance = { amount: 10_000_000, currency: 'DKK' };
DisplayAmount(dkk); // => "10.00 KR"

const usd: Balance = { amount: 10_000_000, currency: 'USD' };
DisplayAmount(usd); // => "$10.00"
```

## Special currencies: XGC and XSC

Two currencies are specific to the Stake US social casino jurisdiction.

**XGC — Gold Coin** (`symbol: GC`, `decimals: 0`)

Gold Coins are the primary play currency in the Stake US social casino. They are whole numbers — `DisplayAmount` shows no decimal places by default.

```typescript
const gc: Balance = { amount: 5_000_000, currency: 'XGC' };
DisplayAmount(gc); // => "5 GC"
```

**XSC — Stake Cash** (`symbol: SC`, `decimals: 2`)

Stake Cash is the redeemable currency in the Stake US social casino. It uses 2 decimal places.

```typescript
const sc: Balance = { amount: 2_500_000, currency: 'XSC' };
DisplayAmount(sc); // => "2.50 SC"
```

<Note>
  When `client.jurisdictionFlags.socialCasino` is `true`, the player is in the Stake US social casino context and will have XGC or XSC balances. Check this flag before displaying any real-money UI elements.
</Note>

## Displaying bet levels

Bet levels returned by `Authenticate()` are API-encoded integers. Use `trimDecimalForIntegers` to display clean whole-number bets without trailing `.00`.

```typescript
import { DisplayAmount, ParseAmount } from 'stake-engine';
import type { Balance } from 'stake-engine';

// authData.config.betLevels = [500000, 1000000, 2500000, 5000000]

authData.config.betLevels.map((level) => {
  const display: Balance = { amount: level, currency: authData.balance.currency };
  return DisplayAmount(display, { trimDecimalForIntegers: true });
});
// => ["$0.50", "$1", "$2.50", "$5"]
```
