## What is Stake Engine?

Stake Engine is a Remote Game Server (RGS) platform for casino game development. The `stake-engine` TypeScript client gives your game frontend a type-safe interface to authenticate players, manage game rounds, track balances, and send game events — all through a single factory function.

## Minimum example

```typescript
import { RGSClient } from 'stake-engine';

const client = RGSClient({ url: window.location.href });

// Authenticate the session
const { balance, config, jurisdictionFlags } = await client.Authenticate();

// Place a bet
const result = await client.Play({ amount: config.betLevels[0], mode: 'base' });

// Complete the round
await client.EndRound();
```

<Note>
  All amounts in the Stake Engine API are integers scaled by `1,000,000`. Use `ParseAmount` and `DisplayAmount` to convert and format them for display. See the [Helpers reference](/api/helpers) for details.
</Note>
